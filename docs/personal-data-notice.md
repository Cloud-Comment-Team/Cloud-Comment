# Уведомление об обработке персональных данных CloudComment (черновик)

> Черновик для MVP. Требует финальной юридической редактуры.

**Версия:** 2026-07-01

## Оператор

CloudComment — сервис встраиваемых комментариев для сайтов.

## Категории персональных данных

1. **Данные аккаунта** — email, хеш пароля, роли пользователя.
2. **Данные авторизации** — сессионные токены (в виде хешей), срок действия, факт отзыва.
3. **Данные комментариев** — текст, URL страницы, идентификатор сайта, автор, время создания.
4. **Данные модерации** — статус комментария, решения модераторов, причины отклонения.
5. **Данные безопасности** — allowed origins, технические метаданные запросов.
6. **Согласия** — версии принятых документов, время и источник регистрации (admin/widget).

## Цели обработки

| Цель | Описание |
| --- | --- |
| **Аккаунт** | Регистрация, хранение учётной записи, назначение ролей |
| **Авторизация** | Вход в админ-панель и виджет, управление сессиями |
| **Комментарии** | Приём, хранение и отображение пользовательского контента |
| **Модерация** | Проверка и публикация/отклонение комментариев |
| **Безопасность** | Защита сервиса, ограничение доступа, предотвращение злоупотреблений |

## Срок хранения

Данные хранятся пока действует аккаунт и необходимы для целей обработки. После удаления аккаунта персональные данные анонимизируются или удаляются в разумный срок, за исключением данных, которые необходимо хранить по закону.

## Права субъекта данных

- получить информацию об обработке;
- исправить неточные данные;
- запросить удаление аккаунта;
- отозвать согласие (может ограничить доступ к сервису).

## data-export {#data-export}

### Как запросить экспорт данных

1. Войдите в админ-панель CloudComment под своим аккаунтом.
2. Откройте раздел **Аккаунт и конфиденциальность**.
3. Направьте запрос на экспорт на **privacy@cloudcomment.example** с email вашего аккаунта.

В экспорт включаются: данные профиля (email, роли, даты), история согласий, комментарии, связанные с вашим аккаунтом, и метаданные активных заявок на удаление.

Срок ответа — до 30 календарных дней (черновик SLA для MVP).
